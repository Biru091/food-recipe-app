# Food Recipe App

## Overview

This is a React-based Food Recipe application that allows users to search and explore different meals using the TheMealDB API. Users can view detailed recipes including images, ingredients, and cooking instructions.

---

## Features

- Search recipes by name
- View detailed recipe information
- Dynamic routing for recipe pages
- Responsive card-based UI
- Context API for global state management
- Live data fetching from TheMealDB API

---

## Tech Stack

- React.js
- React Router DOM
- Context API
- Axios
- CSS (Custom styling)

---

## Key Concepts Used

- useState and useEffect
- useContext (global state)
- useParams (dynamic routing)
- useNavigate (navigation)
- API integration with Axios
